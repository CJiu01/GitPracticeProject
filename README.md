# Git Practice Project

## How to write Commit Message && Pull Request Message

커밋 메시지나 PR 타이틀을 작성할 때 아래의 예시에 있는 것과 같이 \
대괄호를 열고 태그를 작성해주세요 😆 😆

1. 파일 추가 : Add
2. 버그 수정 : Fix
3. 리팩터링 : Refactoring
4. 주석 추가 : Comment
5. 파일 삭제 : Remove
6. 기능 추가 : Feat
7. 문서 수정 : Docs

> Ex) [Feat] Len 함수 기능 완성
